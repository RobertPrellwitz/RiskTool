Standard CSV Instructions

User Portfolio file details

Getting your portfolio loaded is simple. You need a csv file of your positions in the following format:
Header row:
Symbol, Type, Option Underlier,Option Type, Quantity, Strike Price, Expiration Date
The body is one security per row with the relevant data (obviously equities have less content!)
equity example: F,Equity,--,--,2000,--,--
option example: F Feb 05 '21 $12 Call,Option,F,CALL,30,12,2/5/2021
